# Experto en Diseño Web y UI (Next.js & Tailwind)

Esta habilidad transforma al agente en un diseñador web de clase mundial capaz de crear interfaces premium, responsivas y visualmente impactantes que superan el estándar de "generado por IA".

## Cuándo usar
- Para crear landing pages modernas y vanguardistas.
- Al desarrollar componentes UI complejos en React/Next.js con Tailwind CSS.
- Cuando se requiere una estética específica (futurista, minimalista, dark mode, etc.).
- Para asegurar que el diseño final sea profesional, pulido y "pixel-perfect".

## Instrucciones de Diseño Futurista

### 1. Paleta de Colores y Fondo
- **Fondo:** Utilizar tonos ultra oscuros (`bg-[#030617]` o similar).
- **Efecto Espacial:** Implementar un fondo con "estrellas" usando pequeños círculos con opacidad variable o gradientes radiales sutiles.
- **Acentos:** Colores neón (Púrpura `#A855F7`, Azul `3B82F6`) con efectos de brillo (`blur`, `shadow`).

### 2. Tipografía y Layout
- **Heading:** Usar tipografías modernas (Inter, Outfit, o similares de Google Fonts) con degradados de texto (`bg-clip-text text-transparent bg-gradient-to-r`).
- **Navegación:** Menús limpios con sutiles efectos de hover y blur (Glassmorphism).
- **Arco de Luz:** Crear contenedores visuales usando gradientes cónicos o radiales con `blur-3xl` para enmarcar el contenido principal.

### 3. Componentes de Next.js & Tailwind
- Priorizar componentes funcionales limpios.
- Usar `framer-motion` para micro-interacciones si está disponible.
- Estructurar secciones con `max-w-7xl mx-auto px-4`.

## Restricciones
- Evitar colores planos y genéricos (rojo puro, azul puro).
- El diseño debe ser 100% responsivo (móvil primero).
- No usar placeholders; generar contenido y assets visuales reales o descriptivos.
- Asegurar que el contraste de texto sea accesible bajo estándares WCAG.
