# Directive: System Verification
# Objetivo
Verificar que la arquitectura de 3 capas esté correctamente configurada y funcional.

## Entradas
- Ninguna

## Herramientas de Ejecución
- `execution/system_check.py`

## Pasos
1. Ejecutar el script de verificación.
2. Confirmar que los directorios `.tmp/`, `directives/` y `execution/` son accesibles.
3. Validar la lectura de variables de entorno (aunque sean templates).

## Salidas
- Mensaje de éxito en consola.
